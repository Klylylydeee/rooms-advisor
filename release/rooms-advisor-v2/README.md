## Project setup
```
Run 'npm install' to frontend and backend folder.
```
### Web Deployment
```
Please visit http://roomadvisor.herokuapp.com
```
### Local Deployment
```
Run 'mongod' & 'npm start' to backend folder.
```
```
Run 'ng serve' to frontend folder.
```
